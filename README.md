# Comidas Rama Garner 2

Public signup page for missionary meal appointments for Rama Garner 2.

Files included:

- `index.html`, public scheduling page
- `icon-todos.png`, icon for all missionaries
- `icon-elderes.png`, icon for elders
- `icon-hermanas.jpg`, icon for sisters

This site connects to the Garner 2 Google Apps Script Booking API.
